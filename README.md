# Personal Website

A minimalist personal website inspired by modern, clean design aesthetics. Built with pure HTML and CSS—no frameworks, no dependencies, just fast and simple.

## 🚀 Quick Start

1. **Customize the content** in `index.html`:
   - Replace placeholder text with your information
   - Update your name, tagline, and bio
   - Add your professional experience
   - Update contact links with your actual profiles

2. **Preview locally**:
   - Simply open `index.html` in your browser
   - Or use a local server: `python -m http.server 8000`

3. **Deploy to GitHub Pages**:
   - Create a repository named `ahmedhajahmed.github.io`
   - Push these files to the repository
   - Your site will be live at `https://ahmedhajahmed.github.io`

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - Minimalist styling
- `README.md` - This file

## ✏️ Customization Guide

### Update Your Information

Open `index.html` and replace the placeholder content:

1. **Header Section**:
   - Update your name in the `<h1>` tag
   - Change the tagline to reflect your personal statement

2. **Bio Section**:
   - Write your introduction
   - Share what drives you professionally

3. **Experience Section**:
   - Add your work experience, education, or projects
   - Include role titles, companies, and time periods
   - Add or remove experience items as needed

4. **Contact Links**:
   - Update the email link with your email address
   - Add your GitHub, LinkedIn, Twitter, or other social profiles
   - Remove any links you don't want to include

### Modify the Design

Edit `styles.css` to customize:

- **Colors**: Change CSS variables in `:root`
- **Typography**: Adjust font sizes and line heights
- **Spacing**: Modify margins and padding
- **Layout**: Change `--max-width` for wider/narrower content

## 🌐 Deployment to GitHub Pages

### Option 1: New Repository

```bash
# Initialize git (if not already done)
git init

# Add files
git add .
git commit -m "Initial commit"

# Create repository on GitHub named 'ahmedhajahmed.github.io'
# Then push:
git remote add origin https://github.com/ahmedhajahmed/ahmedhajahmed.github.io.git
git branch -M main
git push -u origin main
```

### Option 2: Existing Repository

```bash
# If you already have the repository
git add .
git commit -m "Update website"
git push
```

Your site will be automatically published at `https://ahmedhajahmed.github.io`

### GitHub Pages Settings

1. Go to your repository settings
2. Navigate to "Pages" section
3. Ensure source is set to "Deploy from branch: main"
4. Your site should be live within a few minutes
